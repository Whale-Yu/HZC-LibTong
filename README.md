# 湖院实验通 (HZC LibTong)

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Platform](https://img.shields.io/badge/platform-Chrome-red.svg)

> 一款专为湖州学院实验室准入考试设计的题库搜索工具(湖院实验通)

## 🌟 简介

湖院实验通是一个Chrome扩展程序，可以在指定网页上显示悬浮搜索窗口，帮助学生快速查找实验室安全准入考试的相关题目和答案。该工具支持模糊匹配，实时显示题目、选项和答案，提高复习效率。

## 🎯 功能特点

- 悬浮窗设计，不遮挡网页内容
- 支持模糊搜索，输入关键词即可查找相关题目
- 实时显示题目、选项和答案
- 仅在指定网站运行，安全可靠
- 题型标识清晰，便于区分单选、多选和判断题
- 针对不同题型提供智能复制功能，方便答题
- 自动提取功能，可每秒自动识别题目
- 快速刷题功能，可自动随机选择答案并跳转下一题
- 自动考试功能，可自动匹配题目并选择正确答案
- 美化界面，提升用户体验

## 🚀 安装与使用

### 安装步骤

1. 下载或克隆本项目到本地
2. 打开 Chrome 浏览器，进入 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目中的 `HZC_LibTong` 文件夹

### 使用方法

1. 访问指定网站: `https://zysys.zjhzu.edu.cn/`
2. 页面右上角会出现一个题库搜索助手悬浮窗
3. 在输入框中输入你想查找的题目关键词
4. 点击"搜索"按钮，系统会立即显示匹配的题目
5. 查看结果，包含题目的完整信息，包括题型、题目内容、选项和答案

## ⚡ 新功能说明

### 自动提取题目功能
- **自动提取**：开启"自动提取"开关后，系统会每秒自动识别当前页面的题目并搜索答案，无需手动操作。
- **手动提取**：点击"手动提取题目"按钮，系统会立即识别当前题目并搜索答案。

### 快速刷题功能
- **快速刷题**：开启"快速刷题"开关后，系统会自动随机选择答案并点击"下一题"按钮，帮助你快速完成练习。
- **停止刷题**：关闭开关可停止自动刷题功能。

### 自动考试功能
- **自动考试**：开启"自动考试"开关后，系统会自动匹配当前题目并在题库中查找正确答案，然后自动选择正确选项并点击"下一题"按钮。
- **停止考试**：关闭开关可停止自动考试功能。

## 🔧 技术架构

- 基于 Chrome Extension Manifest V3
- 使用 JavaScript、HTML 和 CSS 实现
- 包含内容脚本(content scripts)用于在目标网站注入功能
- 使用 JSON 格式存储题库数据

## 📁 项目结构

```
HZC_LibTong/
├── HZC_LibTong/           # Chrome扩展主目录
│   ├── manifest.json      # 扩展配置文件
│   ├── content.js         # 主要功能实现
│   ├── floating.css       # 悬浮窗样式
│   ├── questions.json     # 题库数据
│   ├── instructions.html  # 使用说明
│   └── icon.png           # 扩展图标
├── Get_questions/         # 题库爬虫脚本
│   ├── main.py            # 爬虫主程序
│   └── *.csv              # 爬取的原始数据
└── README.md              # 项目说明文档
```
